TimeTable
=========

Simple timetable project.

Files added:

- `timetable.html` — interactive timetable page (open in a browser).
- `AI_INSTRUCTIONS.md` — instructions for AI agents to record requests and summaries.
- `ai-change-log.txt` — append-only change log where agents record timestamped entries.

How to use

- Open [timetable.html](timetable.html) in a browser.
- AI agents should follow `AI_INSTRUCTIONS.md` and append entries to `ai-change-log.txt`.
